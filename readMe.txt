///fun tips for productivity in VScode///



cmd + shft + T = opens a previously closed tab

cmd + shft + E = opens the file explorer 

cmd + downArrowKey = opens the file once in the file explorer
    cmd + W = closes file 

cmd + Enter = opens files to the side 

cmd + del = deletes a file 

cmd + \ = to split the file to the side that were currently working on 



code shortcuts //



HOLD option + arrow left / right key = jumps from word to word 

HOLD option + arrow up / down key = moves the line up or down 

HOLD shift / option + up / down arrow key = copies line up or down 

cmd + enter = inserts a blank new line of code no matter where you are within your codebase
